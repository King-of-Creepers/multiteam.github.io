Pyro table functionality

introducing the gui of the pyrotechnist table.
the numbers:
1-first set of input slot "input markers"
2-second set of input slot "input markers"
3-input set changing buttons
4-input slots
5-type selector board (like in the loom)
6-output slot
7-type buttons (like in the loom)
How it will work: the input set changing buttons will be above the input slots. click on one of them to switch between modes (sets) first set: if the player place a gunpowder into the first input slot, the "type buttons" will show up, and will display all possible fireworkcharge item in the game. you can hover over it, and will display the charge's infos in a form of a tooltip. and if you click on one of them, it will place the required items into the slots, if you have missing items, it will place the items in you have, but it will mark the missing items with red. like when you using the recipe book in crafting, but you have missing items. second set: you only need to place in the charge, a paper, and minimum 1 gunpowder. the other 2 is optional. In this "mode", there will be no "type buttons"

Ghast plant

a very creepy plant, the ghastoid. its the "block version" of the Ghast tear seed. takes 8 minecraft days to grow fully (from stage 1 to stage 8)
Every stage has its drop:
stage 1: drops the seed back
stage 2: same
stage 3: it wont drop the seed back, so you just loose it.
stage 4-5: still dropps nothing
stage 6: when you broke it, it will spawn a ghast.
stage 7: drops (always) 1 ghast tear.
Stage 8 is a kind of corrupted form. the plant will become a ghast spawner. but these ghast will not drop tears or gunpowder, only some xp. if you break it, it will drop a seed with a chance of 20%.

Beehive Gui

1.15.2

The beehive gui.
-it show the honey level in it, and you can take it when its full
-Shows the bees. exactyl:
there is 3 bee slots. the bee icon shows up when that slot is occupied (the icons are those big cute bee "heads").
next to them, you can see informations about the bees. i will go from top to bottom and left to right with explanation:
so one bee has 2 icons. the left one is always changing. if a bee is wandering and just does nothing, it shows no icon. if a bee hading towards a flower, it shows that "> Flower " icon. its a cornflower because thats my favorite flower. if a bee is being pollenated, and hading towards the hive, it shows that "< Flower" icon. the next one i explain is the right one. its a hive icon with either an X mark or a checkmark. it show when that be is being in the hive or flying outside. (checkmar=bee in, X=bee out).
and the last icon really depends on the previous: the honey comb icon. it appears almost everytime when a bee is inside the hive, and marks that the bee is making honey currently.
there should be a lock icon on the bottle input until there's a campfire under the hive so the player doesn't accidentally shift-click and get a bunch of bees following them.
